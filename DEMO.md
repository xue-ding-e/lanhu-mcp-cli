## 快速体验

想亲自体验？只需 3 步：

```bash
# 1. 克隆项目
git clone https://github.com/xue-ding-e/lanhu-mcp-cli.git

# 2. 快速启动
cd lanhu-mcp-cli
./quickstart.sh  # Mac/Linux
# 或 quickstart.bat  # Windows

# 3. 配置 Cursor
# 在 MCP 配置中添加（建议使用英文参数）：
{
  "mcpServers": {
    "lanhu": {
      "url": "http://localhost:8000/mcp?role=Developer&name=YourName"
    }
  }
}
# 注意：部分 AI 工具不支持 URL 中文参数
```

开始对话：
```
帮我看看这个需求文档：[粘贴蓝湖URL]
```

就是这么简单！🎉
